2) Transform And PreDraw
	i)   save();
	ii)  translate(halfWidth, halfHeight);
	iii) translate(-1 * cameraX, -1* cameraY);
	iv)  scale(cameraScale, cameraScale);
	v)   foreach entity in hierarchy:
			a.  wrapper.save();
			b.  call any transform methods with: wrappedContext
			c.  get snapShot
			d.  call any preDraw methods with: snapShot, it will return a bounding polygon in global space
			e.  test return polygon points for existing within camera viewport
			f.  if polygon intersects with viewport: add draw to drawObj at layer
			g.  if children exist:
					wrapper.save();
					loop through children same way
					wrapper.restore();
			h.  wrapper.restore();			
3) Loop through layers and call draw, if draw returns a polygon, push it on stack for mouse events
4) Mouse Events:
    i)   get mouse coords and inverse transform by camera // current in wrapper
    ii)  pop and analyze polys until mouse is contained
    iii) check for enter / exit / click
    iv)  call methods on entity
5) restore();
